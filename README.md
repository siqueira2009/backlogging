Backlogging é uma plataforma para **gamers** organizarem suas bibliotecas de jogos em um só lugar. Ela permite vincular contas de plataformas como Steam (e futuramente Epic Games) para visualizar todos os seus jogos, montar uma lista de pendências (backlog), avaliar os jogos jogados e, futuramente, fazer backup de saves.

O projeto nasce como um site web, com planos de expansão para aplicativos mobile (iOS e Android).
